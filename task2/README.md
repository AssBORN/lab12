# Модульные тесты для CRUD приложения

В данной папке содержатся модульные тесты для приложения управления товарами из `task1`.

## Описание тестов
Тесты написаны с использованием `pytest` и `FastAPI TestClient`. Они покрывают следующие сценарии:
- **Создание**: успешное создание товара и проверка валидации данных (422 ошибка).
- **Чтение**: получение списка всех товаров и поиск конкретного товара по ID (включая случай 404).
- **Обновление**: частичное обновление данных товара и проверка обработки несуществующего ID.
- **Удаление**: успешное удаление товара и проверка обработки несуществующего ID.

## Как запустить тесты
Для запуска тестов необходимо находиться в корневой директории проекта и выполнить команду:
```bash
pytest task2/test_app.py
```
