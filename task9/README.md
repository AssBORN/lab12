# Аналитический отчет по эффективности продавцов

В данной папке реализовано задание по генерации сложного SQL-запроса для анализа деятельности продавцов на маркетплейсе.

## Содержимое
- `report_query.sql`: SQL-запрос для формирования отчета с использованием CTE и агрегатных функций.
- `explanation.md`: Подробный разбор логики работы запроса и рекомендации по оптимизации производительности (индексы).
