# API для управления товарами

REST API для управления товарами на платформе маркетплейса, созданный с помощью FastAPI.

## Особенности
- CRUD операции для Товаров.
- Валидация через Pydantic.
- Хранение данных в памяти для демонстрации.
- Аннотации типов и обработка ошибок.

## Как запустить
1. Установите зависимости:
   ```bash
   pip install fastapi uvicorn pytest httpx
   ```
2. Запустите сервер:
   ```bash
   uvicorn main:app --reload
   ```
3. Доступ к Swagger UI по адресу: `http://127.0.0.1:8000/docs`

## Запуск тестов
Запустите self-тесты, выполнив файл напрямую:
```bash
python main.py
```
