# Валидация артикулов товаров (SKU)

Данное задание реализует проверку формата уникального артикула (SKU) для товаров маркетплейса.

## Правила формата SKU
Артикул должен строго соответствовать следующему шаблону:
`[3 заглавные буквы]-[4 цифры]-[6 цифр]`

- **Код категории**: Ровно 3 заглавные латинские буквы (A-Z).
- **Год**: Ровно 4 цифры (0-9).
- **Порядковый номер**: Ровно 6 цифр (0-9).

**Пример валидного артикула**: `ELE-2026-004512`

## Реализация
Для валидации используется регулярное выражение:
`^[A-Z]{3}-\d{4}-\d{6}$`

- `^` и `$` — якоря начала и конца строки, чтобы исключить частичные совпадения.
- `[A-Z]{3}` — три заглавные буквы.
- `-` — литерал дефиса.
- `\d{4}` — четыре цифры.
- `\d{6}` — шесть цифр.

Для проверки работы валидатора выполните скрипт:
