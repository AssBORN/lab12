### Содержимое репозитория

- `task1` — REST API на FastAPI для управления товарами.
  - `main.py` — реализация CRUD-операций, Pydantic-валидация, встроенные self-tests.
  - `README.md` — инструкция по запуску и тестированию.

- `task2` — модульное тестирование приложения.
  - `test_app.py` — комплексные тесты на `pytest`, покрывающие основные маршруты и граничные случаи.
  - `README.md` — описание тестового покрытия.

- `task3` — рефакторинг бизнес-логики расчета стоимости.
  - `main.py` — оптимизированная функция расчета с использованием словарей-констант и типизации.
  - `README.md` — детальный разбор внесенных изменений.

- `task4` — контейнеризация приложения с БД PostgreSQL.
  - `main.py` — интеграция с SQLAlchemy и PostgreSQL.
  - `Dockerfile` & `docker-compose.yml` — развертывание API и БД.
  - `alembic/` — конфигурация и скрипты миграций базы данных.
  - встроенные self-tests для проверки работы в Docker-окружении.

- `task5` — анализ и аудит бизнес-логики.
  - `analysis.md` — объяснение алгоритма расчета цены и предложения по техническому улучшению.

- `task6` — полная техническая документация проекта.
  - `README.md` — детальный гайд по установке, запуску и API-справочник с примерами.

- `task7` — проектирование расширенной схемы базы данных.
  - `migrations/` — raw SQL скрипты создания таблиц (sellers, products, orders, reviews) и индексов.
  - Реализация связей One-to-Many и Many-to-Many.

- `task8` — аудит безопасности и устранение уязвимостей.
  - `security_report.md` — отчет о найденных уязвимостях (Auth, CORS, XSS).
  - `fixes.md` — описание исправлений (внедрение API-ключей, CORS Middleware, HTML sanitization).

- `task9` — аналитическая отчетность на языке SQL.
  - `report_query.sql` — сложный запрос с CTE для расчета эффективности продавцов.
  - `explanation.md` — разбор логики агрегации и рекомендации по индексации.

- `task10` — валидация артикулов (SKU) с помощью регулярных выражений.
  - `sku_validator.py` — скрипт проверки формата артикула `[A-Z]{3}-\d{4}-\d{6}` с набором тестов.
