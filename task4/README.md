# Docker-развертывание FastAPI приложения

В данной папке находятся конфигурации для запуска приложения в Docker.

## Состав задания
- `Dockerfile`: Сборка приложения на базе Python 3.11-slim с установкой системных зависимостей для `psycopg2`.
- `docker-compose.yml`: Оркестрация двух сервисов:
  - `db`: База данных PostgreSQL 15.
  - `app`: FastAPI приложение, которое зависит от состояния базы данных.
- `requirements.txt`: Список необходимых Python-библиотек.
- `alembic/`: Конфигурация и миграции базы данных.

## Переменные окружения
Приложение использует следующие переменные для конфигурации:
- `DATABASE_URL`: Строка подключения к БД (например, `postgresql://user:password@db:5432/marketplace`).

## Запуск приложения
Для запуска всего стека выполните команду в корне проекта:
```bash
docker-compose -f task4/docker-compose.yml up --build
```

Приложение будет доступно по адресу: `http://localhost:8000`
Swagger UI: `http://localhost:8000/docs`

## Миграции
Миграции выполняются автоматически при старте контейнера с помощью команды `alembic upgrade head` в `CMD` Dockerfile.
